Enrichment and SOC usage

CMDB and TI are applied during writing, then reused in Event Console and incident correlation.

CMDB
Maps observed hosts or IPs to owner, criticality, environment and service. These fields become searchable in events_view.
Threat intel
Matches IPs, hosts, users, processes and raw substrings. Matching events receive searchable fields such as ti_indicator.
Port monitoring
Firewall blocks and SSH events are normalized with target-port context so analysts can see which services are being targeted.
